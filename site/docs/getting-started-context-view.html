Getting Started

Set up Context View

Context View is a landscape read-out of your current focus, timer, and up-next queue — built for a TV, a spare monitor, or the desktop companion's Desk View, not for holding.

📍 Tabby Sidecar, opened on a large landscape screen
1

Trigger it automatically

Open the Sidecar URL on a large landscape screen (roughly 900px+ wide and wider than tall) and it switches to Context View on its own.

2

Or force it from any device

Add ?view=context to the Sidecar URL — useful for casting a phone browser to a TV, or testing the layout on a laptop.

3

Read the screen

Shift state top-left (On shift / On break / Off the clock), the day countdown top-right (if enabled), the current time bottom-middle, your focus + a circular timer centered, and an "UP NEXT" list of what's queued.

4

Get back to the normal view

Tap "Use controls →" on the Context View screen.

5

Customize what it shows

From the extension: Settings → 📺 Context View — day countdown, up next, timeline, checkpoint preview, and the day-reset hour. Full list: Settings & customization.
