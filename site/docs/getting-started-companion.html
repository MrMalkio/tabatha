Getting Started

Install the Desktop Companion

The Desktop Companion is a Windows tray app that watches which apps and windows you're in, so the hours you spend outside the browser stop reading as idle. It also runs Desk View, a standing Context View window for a second monitor.

📍 Windows 10/11 — a tray app, "Tabby Desk"
🚧 The installer is currently unsigned, so Windows SmartScreen will warn "Windows protected your PC" the first time you run it — click More info, then Run anyway, to continue. That's expected until the app is code-signed, not a sign anything is wrong.
1

Download & install

Grab the Windows installer from the showcase site's Get the Desktop Companion panel, run it, and click through the SmartScreen warning above.

2

Find the tray icon

Tabby Desk lives in the Windows system tray once it's running — everything else here happens from that tray menu.

3

Pair it with the extension

Tray → 🔗 Pair Extension (copy token) copies a token; paste it into the browser extension's Settings → 🖥️ Desktop Activity → Companion Pairing → Save & Pair.

Full walkthrough: Pairing devices.
4

Clock in from the tray

Tray → 🕐 Clock In starts your shift from the desktop — the same shift the extension and Sidecar see, on the same clock.

5

Try Desk View

Tray → 🖥 Desk View opens a standing Context View window; toggle 📌 Desk View Always on Top to keep it pinned above other windows. It's the closest thing this family has to a screensaver — leave it running on a second monitor.

6

Leave it running on login

Tray → ✅ Start on Login is on by default; toggle it off there if you'd rather launch Tabby Desk manually.
