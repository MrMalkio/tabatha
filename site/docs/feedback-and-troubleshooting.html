Feedback & troubleshooting

Feedback goes to the same place no matter which surface you send it from. Most cross-device weirdness has one root cause, so check that first.

📍 Extension popup, Tabby Sidecar Settings, and this showcase site
Send feedback
1

From the extension

Toolbar popup → 💬 Send feedback → choose 🐛 Bug or 💡 Idea → describe it → Send. You'll see "✓ Thanks!", or a retry message if it couldn't send.

2

From Tabby Sidecar

Settings → 💬 Feedback & bug report → 🐛 Bug or 💡 Feature request → describe it → Send feedback. Offline, it's saved locally with "📥 Saved locally — we'll send it once the feedback pipeline is live" and a Retry now button once you're back online.

3

About a specific screenshot on this site

Every card on the Show and Tell component gallery has a 🐞 Bug / ✨ Request pair that appears on hover.

Something not working across devices
1

Check sign-in first

Settings → ☁️ Sync & Account on each device. Most cross-device weirdness — a focus that "didn't follow" you, a task that isn't showing up — traces back to being signed into different accounts, or not signed in at all, on one of them.
