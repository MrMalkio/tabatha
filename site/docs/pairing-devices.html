Pairing devices

Two different pairing flows, both short: a 6-digit code for the watch, a copy-paste token for the desktop companion.

📍 Tabby Sidecar Settings, Tabby Watch, and the Desktop Companion tray
Pair a watch
1

Get a code, on your phone

Sidecar → Settings → ⌚ Pair a watch → Get pairing code. It shows a 6-digit code, valid 5 minutes, that works once.

2

Enter it, on the watch

Open Tabby Watch → tap "⌚ Pair with Tabatha" (or "⌚ Re-pair watch" if you've paired before) → enter the 6 digits. It shows "Pairing…" then "Paired ✓."

3

If it fails

"Code invalid or expired — mint a fresh one on your phone" just means the 5 minutes ran out, or it was already used — get a new code and try again.

Pair the Desktop Companion
1

Copy a token, on the desktop

Tabby Desk's tray menu → 🔗 Pair Extension (copy token) copies a pairing token to your clipboard.

2

Paste it, in the browser

Extension Settings → 🖥️ Desktop Activity → Companion Pairing → paste it into Paste pairing token → Save & Pair. You'll see "✓ Token saved — reconnecting to the companion."

3

Is it required?

No — it's a security-hardening step, not a hard requirement. An unpaired companion still connects in open mode; pair it if you want the link authenticated.
