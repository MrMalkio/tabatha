Settings & customization

Most of what's customizable about the family — especially anything a big screen shows — lives in one place: the extension's Settings.

📍 Mostly the extension's Settings; installs happen on each device
Customize Context View
1

Where

Extension Settings → 📺 Context View (sign in first, under Sync & Account — an anonymous profile has nothing to save to).

2

What you can change

Toggle Day countdown, Up next, Timeline, and Show last checkpoint; pick Phone-away fade (Slow fade / Immediate); set "Day resets at (hr)" — the hour (0–23) the day countdown counts down to.

3

When it applies

Changes reach any paired big screen right away — no separate sync step.

Install the apps
1

Tabby Sidecar (PWA)

On your phone: Settings → Account → 📲 Install Tabby (Android/desktop Chrome), or Share → Add to Home Screen on iOS.

2

Desktop Companion

See Install the Desktop Companion — it's an unsigned installer, so expect a SmartScreen prompt on first run.

Set up a standing screen (Desk View)
1

A screensaver-style Context View

Desktop Companion tray → 🖥 Desk View opens Context View in its own window; 📌 Desk View Always on Top keeps it pinned above everything else — the closest thing this family has to a screensaver. Good for a second monitor you glance at all day.
