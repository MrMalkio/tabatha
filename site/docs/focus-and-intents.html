Focus & intents

The unit Tabatha runs on is the intent — a short answer to "what am I doing right now," turned into a focus with a clock. Here's how to start one, close it out, and branch a sub-intent off it.

📍 Extension (Gatekeeper, home dashboard, sidebar) and Tabby Sidecar
Start an intent
1

From a new tab (Extension)

The Gatekeeper overlay asks "Why are you here?" Type into What are you working on? and press Continue for a real focus — or pick ⚔️ Side Quest, 🍬 Sugar Box, 🅿️ Park, or 🔖 Later for anything less than one.

2

From the dashboard or sidebar (Extension)

Type into What are you focusing on? and press Set Focus (dashboard) or ▶ (sidebar).

3

From your phone (Sidecar)

Focus tab → the "🎯 Set focus" card → type into What are you focusing on? (or tap the mic to speak it) → ▶ Set focus.

Resolve a focus
1

Mark it done

Extension: ✓ Resolved on the FocusBar — on the dashboard, the sidebar, or a queue row. Sidecar: ✓ Resolve on the active-focus card, or the ✓ icon on a queue or sub-intent row.

Add a sub-intent
1

Branch off the current focus

Extension: 📌 Sub-focus on the FocusBar → type a name → + Add. Sidecar: the 📌 Sub-intent toggle on the Focus tab → type into Sub-intent under this focus… → Add sub-intent.

2

Work the list

Sub-intents show under their parent with ▶ (switch to it) and ✓ (resolve it) — the parent focus's clock keeps running underneath.

See it The Gatekeeper → Focus & Time components → Tabby Sidecar →
