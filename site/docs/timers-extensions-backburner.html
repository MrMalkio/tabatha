Timers, extensions & backburner

A focus doesn't have to run start-to-finish uninterrupted — pause it, extend it, or set it aside on the backburner while you wait on something else.

📍 Extension InBar/FocusBar, Tabby Sidecar, Tabby Watch
Pause & resume
1

Extension

The InBar's ⏸ button opens "⏸ Where did you leave off?" — leave a note and press Pause. The page turns into a sticky note (📌 Paused) with ✏️ Edit Note and ▶ Resume buttons.

2

Sidecar

⏸ Pause / ▶ Resume on the active-focus card.

3

Watch

The same chip toggles between ⏸ Pause and ▶ Resume.

Extend +5 minutes
1

Any surface

+5m on the extension's FocusBar, the Sidecar's active-focus card, or the watch's ＋5 min chip all add five minutes to the running timer.

When the timer runs out (Extension only)
1

The Focus Timer Expired overlay

Offers ⏱️ Snooze (pick your own minutes), 🍳 Let Me Cook, 🔄 Switch, ⏸ Pause, ☕ Break, ✅ Complete, and 📝 Note. There's no equivalent overlay on Sidecar or Watch today — just extend or pause manually there.

Backburner + snooze
1

Put something aside (Extension)

InBar's 🔥 button opens "🔥 Backburner: '{focus}'" — set a Duration, say what you're waiting for, note what you'll work on instead, then Backburner.

2

Put something aside (Sidecar)

🔥 Backburner button on the active-focus card. The Backburner list shows each item with ▶ (resume), ⏰ (snooze 10 minutes), and ✕ (dismiss).

3

Snooze

Sidecar's backburner rows snooze in a fixed 10-minute step. The extension's Focus Timer Expired overlay lets you snooze by any number of minutes you type in.
