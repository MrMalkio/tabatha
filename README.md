# Tabatha — Context-Driven Tab Manager

**Tabatha** is a Chrome extension designed to help you maintain focus and
organization by enforcing "Context" on your browsing habits. Instead of drowning
in a sea of tabs, Tabatha groups them by what you are _actually doing_ (Context)
and _why_ (Intent).

![Tabatha Icon](icons/icon128.png)

## 🚀 Features (Phase 1.5)

### 🧠 Context & Intent Tracking

- **Context-First Browsing**: Every tab belongs to a "Context" (e.g., "Work",
  "Learning", "Entertainment").
- **Intent Prompt**: Start a new flow? Tabatha asks _what_ you are doing to keep
  you intentional.
- **Smart Detection**: Automatically categorizes URLs (e.g., YouTube →
  Entertainment/Media, GitHub → Learning/Work).

### 📂 Organization

- **Context View**: View your tabs grouped by their context rather than a flat
  list.
- **Groups & Sub-Groups**: sophisticated hierarchy beyond Chrome's native
  groups.
- **Saved Groups**: (In Progress) Save and restore entire contexts to clear your
  workspace without losing state.

### ⏱️ Time & Focus

- **Time Tracking**: Tracks active time per tab, category, and context.
- **Pomodoro Timer**: Integrated Focus/Break timers to manage your energy.
- **Idle Detection**: Detects when you leave Chrome and asks for your "Offline
  Context" when you return.

### 🔒 Tab Locking

- **Lock**: Prevent accidental closing of important tabs.
- **URL Lock**: (Optional) Lock a tab to a specific URL/Domain; navigating away
  opens a new tab instead.

## 🛠️ Installation (Developer Mode)

1. Clone this repository:
   ```bash
   git clone https://github.com/mrmalkio/Tabatha.git
   ```
2. Open Chrome and navigate to `chrome://extensions`.
3. Enable **Developer mode** (top right toggle).
4. Click **Load unpacked**.
5. Select the `Tabatha` directory.
