# C12 — Team / Onboarding SOP Mode

Status: stub — Fable to expand

Phase: Phase 4

## Purpose
Extends Cortex from an individual optimizer to a team capability. Managers point Cortex at new hires to shorten SOP ramp-up, and admins can mandate capture-on-clock-in so org-level observation is enforceable and consistent.

## Key behaviors
- **Onboarding acceleration** — after observing a new hire's workflow, Cortex suggests tailored extensions/hotkeys → org-scale time savings.
- **SOP learning** — new hires learn standard operating procedures faster from observed exemplar workflows.
- **Org mandate** — admins can require capture-on-clock-in for team members, enforced via org policy + `browser_profiles`/awareness.

## Dependencies
- C3 (Storage & Retention Fabric) — org partition + admin-set retention.
- C7 (Recommendation & Action Layer) — delivers tailored hotkey/extension suggestions.
- C6 (Optimization Loop) — runs the analysis over team ledgers.
- C14 (Agent Data Map & Governance) — org data governance.
