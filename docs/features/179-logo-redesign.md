# Task #179 — Logo Redesign

> **Status:** 📋 Planned · **Version:** v0.2.0  
> **Type:** Design Task (not a software feature)  
> **Created:** 2026-05-15

## User Context (Quotes)

> "We need a new logo for Tabatha — this one looks creepy in a 'I'm watching you' kind of way and also kind of phallic. We don't want that."
> — User, 2026-05-15

## Problem

The current Tabatha logo has unintended negative associations:
- **"I'm watching you" vibe** — feels surveillance-like, contrary to Tabatha's philosophy of *self-awareness* and *intentionality*
- **Phallic resemblance** — unintentional shape issue that needs to be corrected

## Design Direction

The new logo should convey:
- **Intentionality** — mindful, purposeful, not surveillant
- **Warmth** — approachable, friendly, empowering
- **Focus** — clarity, direction, attention
- **Flow** — smooth, dynamic, in-the-zone feeling
- **Part of Flux ecosystem** — visual consistency with Flux branding

### What It Should NOT Be

- ❌ Eye/surveillance imagery
- ❌ Overly geometric/cold/corporate
- ❌ Shapes with unintended anatomical resemblance
- ❌ Too complex for favicon/16px rendering

### Concepts to Explore

- Abstract flow/wave mark (attention as a wave)
- Stylized "T" lettermark
- Focus lens/prism (light being directed, not watched)
- Compass/north star (direction without surveillance)
- Abstract brain/neural spark (cognition, not watching)

## Deliverables

- Extension toolbar icon (16×16, 32×32, 48×48, 128×128)
- Favicon
- Full logo with wordmark for website/docs
- Dark mode variant
- Monochrome variant for small sizes
