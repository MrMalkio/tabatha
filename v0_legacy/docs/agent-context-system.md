# Tabatha — AI Agent Context System

> How Tabatha packages browsing activity for AI agents automatically.

---

## Overview

Tabatha generates a **Context File** — a structured Markdown document containing your browsing history, time data, and intent metadata — and deposits it where your AI agents can find it. This gives every AI you use persistent knowledge of your work patterns without manual copy-paste.

**Philosophy**: Tabatha works 100% without AI. But when paired with an agent, it becomes a **shared memory layer** between you and your AI tools.

## Agent Instruction Line

When Tabatha injects a context file, it also appends a single instruction line to the agent's memory/rules file:

```markdown
<!-- TABATHA CONTEXT: This workspace user's browsing activity is documented in 
tabatha-context.md (updated every [interval]). Reference this file to understand 
what the user has been researching, their current project contexts, and how they 
spend their time in the browser. For detailed history: ~/.tabatha/history.json -->
```

## Configuration

### User Settings (in extension)

| Setting | Default | Options |
|---------|---------|---------|
| Context file enabled | OFF | ON/OFF |
| Update frequency | 30 min | 5m / 15m / 30m / 1h / 4h / daily / manual |
| Output format | Markdown | Markdown / JSON / YAML |
| Output path | Auto-detect | Custom path or auto-detect agent |
| Agent auto-detect | ON | Scans for `.cursor/`, `.claude/`, `.gemini/` etc. |
| Include URLs | ON | ON/OFF (privacy control) |
| Include time data | ON | ON/OFF |
| Include closed tabs | OFF | ON/OFF |
| History retention | 30 days | 7d / 30d / 90d / unlimited |
| Persist through history clear | ON | ON/OFF (Tabatha keeps its own copy) |

### Data Privacy

- Context file is **local only** — never uploaded anywhere
- User controls exactly what data is included
- URLs can be masked (`github.com/***` instead of full path)
- Private/incognito tabs are **never** included
- User can delete all Tabatha history at any time

## Architecture

```
background.js
  ├── ContextFileGenerator (new module)
  │   ├── collectBrowsingHistory()      ← from chrome.history API
  │   ├── collectTabathMetadata()       ← from chrome.storage.local
  │   ├── collectTimeTracking()         ← from state.timeTracking
  │   ├── formatContextFile()           ← markdown/json/yaml
  │   └── writeToAgentPaths()           ← fs write via downloads API
  │
  ├── AgentDetector (new module)
  │   ├── scanForAgentConfigs()          ← checks known paths
  │   ├── getActiveAgentPaths()          ← returns writable locations
  │   └── injectInstructionLine()        ← appends to agent .md files
  │
  └── HistoryPersistence (new module)
      ├── captureHistorySnapshot()       ← periodic chrome.history.search
      ├── storePersistentHistory()       ← chrome.storage.local (survives clear)
      └── exportHistory()               ← full JSON dump
```
