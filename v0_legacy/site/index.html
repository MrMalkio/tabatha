Current Features

Everything running in Tabatha today

Every feature below is live and working. Tabatha is a lightweight Chrome extension — no desktop app, no signup, no servers.

⏱

Automatic Time Tracking

Active time, open duration, and category breakdown per tab — all tracked silently in the background. No timers to start or stop.

Learn more →
🛡️

Gatekeeper Overlay

"Good Friction" — an interstitial overlay on new tabs that asks your context and intent before you browse. Prevents mindless tab-opening.

🏷️

Context & Intent Engine

Every tab gets a named Context ("Q1 Report") and Intent ("Research competitors"). New tabs inherit context from their parent tab.

📊

Auto-Categorization

9 built-in categories (Work, Media, Learning, etc.) automatically classify every URL. Custom categories supported.

🔒

Tab & URL Locking

Lock a tab to prevent accidental closure. URL Lock jails a tab to its domain — no more "jailbreaking" to YouTube.

🍅

Pomodoro Timer

Built-in 25/5 Pomodoro with system notifications. Focus sessions and breaks managed from the dashboard or sidebar.

😴

Step Away Mode

One-click pause. Screen goes dark, timers pause, and when you return, Tabatha welcomes you back with context recall.

💾

Session Snapshots

Automatic snapshots every 5 minutes. "Return to Flow" lets you restore your exact workspace after a crash or restart.

📝

Markdown Export

One-click export of your entire browsing session as a structured Markdown file — ready for AI agents, journals, or reporting.

🔍

Quick Tab Search

Instant fuzzy search across all open tabs. Switch to any tab in 2 keystrokes from the popup.

📂

Tab Groups & Projects

Organize tabs into Chrome Tab Groups and sub-groups (Projects). Assign priorities from Critical to None.

🤖

Agent-Ready Data

Structured data exports designed for consumption by AI agents. Your browsing context becomes machine-readable intelligence.

🧠

AI Agent Context System

Securely package your browsing history and intent context into a local Markdown file that your AI Agent (Cursor, Claude, Windsurf) can auto-read. Next Release

🔗

Zero-Integration Task Sync

Automatically catalog workspaces, projects, and tasks from URLs (Asana, Jira, etc.) without OAuth. Log browsing sessions directly to assumed tasks.

📷

Smart Capture (Opt-In)

Optional screenshot timeline and keystroke analytics. OFF by default. Enable per-feature for proof-of-work, visual audit, or activity scoring.

🔗

Second Brain Connectors

Drop context files into Obsidian, Logseq, Open Brain (OB1), or any Markdown-based system. Your browsing fuels your knowledge graph.

👥

Pro Mode — Multi-Profile

Sync Tabatha across Chrome profiles. Track clients, projects, and teams. Admin vs. user profiles. Employer visibility controls.

Core Superpower

Time Tracking — Mind-Numbingly Detailed

Tabatha doesn't just track "time on computer." It tracks time per tab, per context, per category, per group — passively, accurately, and privately.

Per-Tab Active Time

Only the foreground tab accumulates active time. Background tabs track open duration separately. You see both.

Per-Category Rollup

Work vs. Media vs. Learning — see the split instantly. 9 auto-assigned categories, or define your own.

Per-Context Aggregation

How long did "Q1 Report" take? Tabatha tells you by summing every tab tagged to that context.

Idle Detection

When you stop interacting, time stops counting. No inflated numbers. Welcome-back notifications when you return.

Context Switch Alerts

Tabatha detects when you jump between unrelated contexts and can nudge you back to your declared intent.

Markdown Timesheets

Export your entire day as a structured Markdown file — with time, contexts, and categories — ready for billing or review.
